# Java Bank Management System

## Description
This is a console-based bank management system developed in Java. The project simulates basic banking operations and demonstrates structured programming and Object-Oriented Programming concepts.

## Features
- Create bank accounts
- Deposit money
- Withdraw money
- Check account balance

## Concepts Used
- Methods
- Arrays
- Conditional statements
- Basic OOP principles

## Technologies
- Java
- Console I/O

## How to Run
1. Compile the Java file
2. Run the main class
3. Follow the instructions in the console
